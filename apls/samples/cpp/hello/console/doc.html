C++ Hello Console Sample

This is the simplist C++ sample that can be run from the command-line. It ensures everything is installed correctly.

Windows

  1. Compile and link hello.pro into hello.xpl.
    1. acmp hello
    2. alnk hello hello
  2. Under Windows, open the Microsoft Visual C++ Project hello and build hello.exe.
  3. Run hello(.exe)

Unix/Mac

  1. Run the appropriate makefile build chello
  2. run: ./chello
