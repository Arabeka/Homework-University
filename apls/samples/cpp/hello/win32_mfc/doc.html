C++ Hello Win32 Sample

This is the simplist C++ sample that can be run from the MS-Windows environment under the Microsoft Foundation Classes. It ensures everything is installed correctly.

Building the Sample

  1. Compile and link hello.pro into hello.xpl.
  2. Under Windows, open the Microsoft Visual C++ Project hello.dsw and build hello.exe.

Running the Sample

  1. Run hello.exe.
