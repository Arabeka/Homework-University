C Pets Sample

This sample demonstrates calling a simple expert system from C.

Building the Sample

  1. Compile and link pets.pro into pets.xpl.
  2. Under Windows, open the Microsoft Visual C Project pets.dsw and build pets.exe.
  3. Under Unix, run the appropriate makefile under Linux (makefile.lnx) or Solaris (makefile.sol) to build pets

Running the Sample

  1. Run pets(.exe).
  2. Enter a sound like 'woof', 'quack' or 'meow'.
